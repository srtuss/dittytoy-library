================
Dittytoy Library
================

**Dittytoy Library** is a collection of DSP and utility classes and functions for `Dittytoy <https://dittytoy.net>`__ to speed up and simplify your music coding. 🎵

The naming style of the library's components favors brevity in the spirit of code-golfing.

Background
==========

The Dittytoy API only offers very basic features for song creation. Whilst this is an exciting limitation for the experienced audio coder to tackle, it may be an impossibly high bar of entry for the newcomer coding enthusiast.

This library aims to add concepts and algorithms popular in the modular synthesizer world to the Dittytoy API together with proper documentation, removing the necessity for users to try and reverse-engineer code from other postings on Dittytoy.

This library is created by Dittytoy user `srtuss <https://dittytoy.net/user/srtuss>`_ and contains original code as well as code adapted from open third-party sources.

Use of the library is granted for non-commercial applications as long as the commented line above the code is included.
